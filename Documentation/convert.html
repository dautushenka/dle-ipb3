Перенос пользователей
Перес пользователь работает в сторону переноса пользователей с ДЛЕ в форум.
Для переноса пользователей с форума на сайт, нужно разрешить вход только с форума и включить в настройках интеграции на форуме (файл dle_config.php) перенос пользователей с форума на сайт при их входе, таким образом пользователи постеменно будут переноситься на сайт без изменения им пароля.

Также этот модуль авторизации отвечает за интеграцию восстановления пароля.При переносе с сайта на форум:
Перес пользователей происходит автоматически при входе пользователя на сайт или на форум. Если пользователь входит на сайт и для него не находиться акаунт на форуме то он автоматически создаётся. Если пользователь заходит на форум и форум не находит у себя акаунта пользователя он делает запрос в ДЛЕ в поисках даных, если такой пользователь находиться то форум его автоматически регистрирует и у себя.Для того чтобы форум автоматически запрашивал акаунт пользователя в ДЛЕ нужно добавить метод аторизации на форуме. Для это идём в админку форума: 1. Настройки и инструменты > 2. Модули авторизации > 3. Управление модулями авторизации > 4. Добавить метод.

Название: DataLife Engine
Описание: Авторизация и конвертирование пользователей ДЛЕ
Имя директории с файлами модуля: dle
Модуль включен?: Да

Остальные поля не заполняются.
