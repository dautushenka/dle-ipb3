DLE + IPB3 v1.0.5
Установка
1. Устанока интеграции производиться только на установленные скрипты ДЛЕ и форума
2. Заливаем все файлы из папки dle_uploads в корень ДЛЕ на сервер в бинарном (двоичном режиме)(binary mode), они имеют такую же сткрутуру каталогов как и DLE. Файлы из папки ipb3_uploads заливаем в корень форума в бинарном (двоичном режиме)(binary mode), они имеют такую же сткрутуру каталогов как и IPB3.
3. Скопировать фай conf_global.php из корня форума в корень ДЛЕ
4. Содержимое папки templates (файлы *.tpl) скопировать в папку с вашим с шаблоном, зарания их подогнать под ваш шаблон, по умолчанию они сделаны для шаблона Default.
5. Зарустить мастер установки http://www.ваш_домен/install.php, далее следуйте его инструкциям. Не переходите к следующим пункам пока не закончите работу с мастером установки.
Изменить файлы можно не все, к каждомк изменению приведены коментарии, если вы не собираетесь использовать ту или иную возможность, можно изменения для него не проводить.
